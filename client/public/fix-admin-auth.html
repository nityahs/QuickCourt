QuickCourt Admin Authentication Fix

Problem

The admin facilities page is showing "Failed to load facilities" because the API requests to /api/admin/facilities are returning 401 Unauthorized errors.

Fix Steps

  1. Click the button below to set a valid admin authentication token in your browser's localStorage
  2. Return to the admin facilities page and refresh
  3. The facilities should now load correctly
Note: This is a temporary fix for development purposes only. In a production environment, proper authentication should be implemented.
